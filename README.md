# 🤖 Никитос AI

Твой персональный AI ассистент в виде PWA-приложения.

## 🚀 Быстрый старт

### 1. Получи API ключ
- Зайди на https://console.anthropic.com
- Создай аккаунт → API Keys → Create Key
- Скопируй ключ (начинается с `sk-ant-...`)

### 2. Запусти приложение
Открой файл `index.html` через локальный сервер (не просто двойным кликом):

```bash
# Python (если установлен)
cd nikitos-ai
python -m http.server 8080

# Или Node.js
npx serve .
```
Затем открой http://localhost:8080 в браузере.

### 3. Введи API ключ
Вкладка **Настройки** → 🔑 API ключ → вставь ключ → Сохранить

### 4. Установи на телефон
- **iPhone (Safari)**: Поделиться → «На экран "Домой"»
- **Android (Chrome)**: Меню → «Установить приложение»

## ✨ Функции

| Функция | Описание |
|---------|---------|
| 💬 Чат | Полноценный чат с AI, история, код, форматирование |
| 🔍 Поиск | Кнопка 🔍 — ищет актуальную информацию в интернете |
| 📎 Файлы | Прикрепляй PDF и картинки, задавай вопросы |
| 🧠 База знаний | Скармливай тексты, URL, файлы — Никитос их запомнит |
| ⚡ Авто-обучение | Задай темы → Никитос сам ищет и учится |
| 🎙️ Голос | Говори вместо печатания, ответы озвучиваются |
| ⚙️ Настройки | Меняй имя, личность, характер ассистента |

## 🧠 Как работает "обучение"

Никитос использует **RAG (Retrieval Augmented Generation)**:
1. Ты добавляешь знания в базу (тексты/URL/файлы)
2. При авто-обучении — Claude сам ищет информацию по твоим темам через веб-поиск
3. Все знания включаются в контекст при каждом ответе
4. Чем больше база знаний — тем умнее Никитос в твоих темах

Это реальная, рабочая система персонализации AI без GPU серверов!
